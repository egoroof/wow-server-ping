# wow-server-ping

| [🇬🇧 English](README.md) | **🇷🇺 Русский** |
| :-: | :-: |

Утилита для пинга серверов World of Warcraft 335a. Может корректно вычислять пинг для серверов за прокси.

![console usage](./images/console.png)

Обозначения:

- `Conn` - среднее арифметическое времени подключения к игровому серверу в миллисекундах
- `Ping` - среднее арифметическое времени пинга игрового сервера в миллисекундах
- `±` - среднее абсолютное отклонение для `Conn` и `Ping`
- `T1` - таймауты во время изначального TCP соединения
- `T2` - таймауты после `T1` и до получения первого сообщения от сервера
- `T3` - таймауты после `T2`, во время отправки сообщения серверу и до закрытия соединения с сервером
- `E` - ошибки

Так же может работать как экспортёр метрик Prometheus и отображать графики в Grafana:

![grafana usage](./images/grafana.png)

## Использование

### Скачивание

Для Windows можно скачать сборку со [страницы релизов](https://github.com/egoroof/wow-server-ping/releases/latest). Если нужны сборки для других ОС - создайте багрепорт.

### Список миров

Если вас интересует сервер `WoW Circle 3.3.5a`, то список миров не нужно извлекать - он уже поставляется вместе со сборкой. Можете переходить к следующий главе.

Сначала вам нужно будет извлечь список миров. Сервера WoW дают список миров только после логина, поэтому вам придётся ввести ваш логин/пароль. Этот проект содержит утилиту `realmlist.exe`, которая логинится на сервер WoW подобно реальному игровому клиенту WoW и сохраняет список миров в папку `servers`.

Использование:

```shell
realmlist.exe [-port N] [-timeout T] user host
```

Пример:

```shell
realmlist.exe admin logon.wowcircle.me
```

Если переживаете за свои учётные данные, то можете запустить Wireshark, залогиниться через свой игровой клиент и самостоятельно извлечь список миров.

### Пинг

Простой пример, который загружает список миров из файла `servers/logon.wowcircle.me.json`, отправляет пинг запросы и выводит статистику каждые 10 секунд:

```shell
wow-ping.exe logon.wowcircle.me
```

Можно отфильтровать список серверов с помощью регулярного выражения опцией `-filter`:

```shell
wow-ping.exe -filter "x4" logon.wowcircle.me
```

Сборки для Windows содержат `.bat` файлы, которые вы можете использовать, либо сделать подобные для себя.

### Доступные настройки

| Флаг | По умолчанию | Описание |
|---|---|---|
| `-port` | - | Порт для метрик Prometheus |
| `-timeout` | `1s` | Таймаут пинга |
| `-interval` | `1s` | Время ожидания между запросами |
| `-stats-interval` | `10s` | Как часто выводить статистику в консоль |
| `-stats` | - | Сколько вывести статистики, прежде чем выйти |
| `-filter` | - | Регулярное выражение для фильтрации серверов по названию |

### Описание процесса пинга

#### Через прокси сервер

Сетевые запросы в процессе одного пинга:

1. Вы -> TCP SYN -> Прокси
2. Прокси -> TCP SYN-ACK -> Вы
3. Вы -> TCP ACK -> Прокси
4. Прокси -> TCP SYN -> Сервер
5. Сервер -> TCP SYN-ACK -> Прокси
6. Прокси -> TCP ACK -> Сервер
7. Сервер -> пакет `SMSG_AUTH_CHALLENGE` -> Прокси -> Вы
8. Вы -> пакет `CMSG_PING` -> Прокси -> Сервер
9. Сервер -> TCP FIN -> Прокси -> Вы
10. Вы -> TCP FIN -> Прокси -> Сервер

Некоторые не важные запросы подтверждения скрыты для простоты.

Время подключения (`Conn`) измеряется в шагах 1 - 2, а пинг (`Ping`) сервера в шагах 8 - 9.

Таймауты могут помочь при отладке потерь сетевых пакетов. Таймауты разделены на 3 типа:

- `T1` - если возникнет в шагах 1 - 2 (вы - прокси)
- `T2` - если возникнет в шагах 3 - 7 (прокси - сервер)
- `T3` - если возникнет в шагах 8 - 9 (вы - сервер)

#### Без прокси сервера

1. Вы -> TCP SYN -> Сервер
2. Сервер -> TCP SYN-ACK -> Вы
3. Вы -> TCP ACK -> Сервер
4. Сервер -> пакет `SMSG_AUTH_CHALLENGE` -> Вы
5. Вы -> пакет `CMSG_PING` -> Сервер
6. Сервер -> TCP FIN -> Вы
7. Вы -> TCP FIN -> Сервер

Время подключения (`Conn`) измеряется в шагах 1 - 2, а пинг (`Ping`) сервера в шагах 5 - 6.

Без прокси сервера значения `Conn` и `Ping` должны быть практически одинаковыми. Но, поскольку мы не знаем о наличии прокси, то выполняем тот же процесс пинга, что и через прокси.

Таймауты:

- `T1` - если возникнет в шагах 1 - 2
- `T2` - если возникнет в шагах 3 - 4
- `T3` - если возникнет в шагах 5 - 6

## Антивирусы

Некоторые антивирусы могут ошибочно помечать скачиваемую сборку для Windows как вирус и блокировать скачивание. Можете добавить исключение и попытаться скачать заново. Эта утилита не содержит вирусов. Можете проверить исходный код и скомпилировать самостоятельно с помощью golang. Кроме того, можете проверить через VirusTotal.
